# Instacart Portfolio — Jessica Batbayar

A two-project portfolio built for Instacart's APM program application.

**Live site:** [jessicabat.github.io/instacart](https://jessicabat.github.io/instacart)

## Projects

**Project A — Catalog-Powered Smart Substitution via Store View**
A prototype safety engine that evaluates whether a candidate substitute is
safe for the consumer given their dietary context.
→ [Project repo](https://github.com/jessicabat/instacart-substitution)
→ [Project page](https://jessicabat.github.io/instacart/project-a)

**Project B — Shopper Earnings at Risk**
A data model quantifying how Cart Assistant-style order complexity affects
shopper hourly earnings, with a three-option policy recommendation.
→ [Project repo](https://github.com/jessicabat/instacart-shopper-model)
→ [Project page](https://jessicabat.github.io/instacart/project-b)

## Stack
HTML, CSS, vanilla JS. Hosted on GitHub Pages.
